HCI & Preservation Architecture

01. High-Fidelity Digitization: This interactive architecture transcends basic web design by treating authentic photographic textures as the primary UI layer. By preserving the 'Tactile Soul' of the Majolica tile, we prevent the historical flattening common in digital archives.


02. Cognitive Affordances: Utilizing principles from advanced Human-Computer Interaction, specific CSS transformation matrices and luminous filters are applied during pointer events. This non-verbal feedback loop subconsciously trains the user to recognize the artifact as an interactive narrative portal.


03. Heuristic Branching Narrative: Moving away from linear pedagogical models, this prototype utilizes a multi-nodal storytelling engine. It requires the user to make ontological choices, transforming them from a passive museum observer into an active historian. This significantly deepens cognitive retention.


04. No-Code & Vibe Coding Paradigms: Developed through LLM-assisted rapid prototyping, this project embodies the democratization of digital humanities. It aligns strictly with recent literature advocating for accessible authoring tools that empower cultural stakeholders to build their own immersive serious games.
